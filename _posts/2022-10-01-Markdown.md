# Markdown Guide

Markdown은 일반 텍스트 문서에 서식 요소를 추가하여 사용할 수 있는 간단한 마크업 언어입니다. 문법이 쉽고 간단한 것이 특징이며, HTML 등 서식 문서로 쉽게 변환되기 때문에 온라인 게시물 등에 많이 사용됩니다.

참고 : [마크다운 가이드](https://www.markdownguide.org/)

***

## 기본 문법

### 1. 제목

- 제목을 만들려면 단어 앞에 # 기호를 추가합니다. 

# H1
## H2
### H3
#### H4
##### H5
###### H6

```
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

### 2. 제목을 만드는 대체 구문


Heading level 1
===============

Heading level 2
---------------

```
Heading level 1
===============

Heading level 2
---------------
```

### 3. 굵게

```
***굵게**
```

>***굵게***
